# Little game

## install

### for linux users (real ones)
```
sudo apt-get install -y build-essential libx11-dev libxext-dev libbsd-dev
git clone https://github.com/vimman/wolf3d
cd wolf3d
git clone https://github.com/vimman/libft
make
```

### for osx users
```
git clone https://github.com/vimman/wolf3d
cd wolf3d
git clone https://github.com/vimman/libft
make
```
